IAR contains unknow tools 解决方法

相关知识：

IAR工程描述文件采用标准的XML形式，各个文件的功能如下：

文件后缀 功能描述
*.ewp IAR工程文件，包含配置项和文件列表
*.ewt IAR配置文件，包含C-STAT和RuntimeChecking的信息
*.dep IAR调试信息配置，主要用于描述源文件和obj文件的关联信息
*.ewd 调试相关，比如选择的调试器和调试器ID等等
*.eww IAR workspace，用于管理多个project

错误描述

出现如下错误：

The project '...' contains the unknown tool 'Coder'

  1. 寻找包含tool Coder配置的文件，可能在.ewp或者.ewd中，如果没有就去其他文件查找。这些文件用记事本打开。
