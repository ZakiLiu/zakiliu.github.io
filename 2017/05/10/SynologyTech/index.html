Synology FTP 目录文件夹更改

需求：新增一个账号，想和同事，朋友，网友分享Synology中已有的目录该怎么办呢？

方法：

  1. 新增使用者帳號 => 使用者帳號 → 新增 (假設叫做anime)

  2. 開啟使用者家目錄 => 使用者帳號 → 進階設定 → 啟動家目錄服務

  3. 更改使用者的FTP根目錄為家目錄 => 檔案服務 → FTP → 進階設定 → 變更下列使用者的根目錄至家目錄
  4. 開啟ssh連線 => 終端機 & SNMP → 啟動SSH功能
  5. SSH登录Synology
  • 路径根据实际情况修改
  • 把指令写到/etc/rc.local的目的為開機時自動mount
