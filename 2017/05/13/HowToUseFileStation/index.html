File Station访问方法

File Station可以通过如下这些方式访问：

  1. 最简单的通过浏览器打开，网址链接管理员提供，浏览器需要Flash插件。
  2. 通过移动端APP访问，iOS在APP Store搜DS File，Android点击DS File下载。

PS:

  1. 使用File Station需要先设置好Synology的File Station，或者管理员给您开放权限后的网址，账户名和密码。
  2. 使用 File Station 通过网页浏览器（如 Internet Explorer）上传文件时，可上传的最大文件为 2GB。如果您要上传大于 2GB 的文件，上传进程将失败。
